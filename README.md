# NRORG262 - Computer Organisation and Architecture Notes

This repository contains notes for my university module **Computer Organisation and Architecture**.

The material covers:

- CPU structure and operation
- Instruction set architecture (ISA)
- Assembly language basics and examples
- Data path and control units
- Memory hierarchy and cache concepts
- Input/output and interrupt handling
- How a computer works: components, buses, and cycles

## Content Overview

- `Assembly/` – Example assembly programs and syntax notes
- `Diagrams/` – Architecture diagrams illustrating how a computer works
- `Notes/` – Organized lecture notes for key topics in the module

## Recommended use

1. Review the diagrams to understand the high-level flow of data and control.
2. Study the lecture notes for concepts such as CPU design, pipelining, and memory systems.
3. Use the assembly examples to practice writing and tracing instructions.

## Scope

These notes are intended to support learning for:

- Digital logic and processor design
- Assembly programming
- System organization and architecture
- Hardware-software interaction
